如果你想要搭一个个人博客，你需要有：
云服务器 一个

以及：
Xshell 软件 下载

服务器

购买一个服务器

如果你是一个大学生，那么你将会享受阿里云和腾讯云的双重优惠。总之腾讯云学生机会实在一些，阿里云的附加业务以及其他API会方便一些。如果仅仅搭博客的话，选择

领取一个代金券，然后去购买服务器时候选择香港地段（为了防止备案）

系统选择Ubuntu 64位最新系统， 记住你的帐号和密码，安全组设置为开放所有端口。

配置服务器

打开Xshell， 文件 ， 新建， 主机填你的服务器IP地址，连接。
输入帐号密码，就进入shell。
sudo apt-get install apache2 安装Apache
不出意外，从你的浏览器访问你的IP地址，就会看到Apache默认页面Ubuntu Logo Apache2 Ubuntu Default Page

sudo apt-get install php 安装php
sudo apt-get install libapache2-mod-php 安装php-mod-sgi
sudo apt-get install php-mysql 安装php-mysql

sudo apt-get install mysql-server 安装Mysql 不出意外会出现让你设置密码的界面，重复两遍即可。

sudo passwd 输入两次密码设置root账户的密码

配置数据库

输入 mysql -u root -p，然后输入密码进入数据库，这个密码是你刚才创建Mysql帐号的密码。

这样你就创建了一个数据库 wp

配置wordpress

安装wordpress

进入apache根目录cd /var/www/html
切换为root用户 su 然后输入root密码，

wget https://cn.wordpress.org/wordpress-4.7.4-zh_CN.tar.gz下载 wordpress
如果国外源速度太慢，可以切换到我上传的国内云wget http://hrohrbucket.oss-cn-hangzhou.aliyuncs.com/wordpress-4.7.4-zh_CN.tar.gz下载。

tar -xzvf wordpress-4.7.4-zh_CN.tar.gz 解压 wordpress
mv wordpress/* ./ 移动wordpress文件到当前目录
rm index.html 删除apache的默认主页
sudo /etc/init.d/apache2 restart重启apache服务器。

不出意外，现在访问youdomain/wp-admin会出现

如下图这样设置，

提交之后会出现
复制里面的所有文字到剪切板，
前往网站根目录，cd /var/www/html, vim wp-config.php出现一个黑屏，此时按 i，然后将所有的东西复制（键盘shift+insert）进去，然后按键盘的Esc，然后输入:wq，保存。

然后安装，进入下一个页面，自己填写，建议不要用随机的密码，还是用一个自己熟悉的密码比较好。

这样我们就完成了Wordpress的安装。
