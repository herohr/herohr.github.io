何为HTTP报文

HTTP是一个“客户端-服务器”的协议，由任意端发起的、发送的数据 称为HTTP报文

报文的组成

HTTP报文是文本组成的，是人类可读的字符。
HTTP报文的的请求头必须以回车换行符（\r\n）来区分每一行，请求头与正文必须也有一个\r\n作为分隔。

如果你想要搭一个个人博客，你需要有：
云服务器 一个
域名 一个
电脑 一台

以及：
Xshell 软件 下载
FlashFXP 软件 下载
一点点智商 下载

服务器

购买一个服务器

如果你是一个大学生，那么你将会享受阿里云和腾讯云的双重优惠。总之腾讯云学生机会实在一些，阿里云的附加业务以及其他API会方便一些。如果仅仅搭博客的话，选择

领取一个代金券，然后去购买服务器时候选择香港地段（为了防止备案）

系统选择Ubuntu 64位最新系统， 记住你的帐号和密码，安全组设置为开放所有端口。

配置服务器

打开Xshell， 文件 ， 新建， 主机填你的服务器IP地址，连接。
输入帐号密码，就进入shell。
sudo apt-get install apache2 安装Apache
不出意外，从你的浏览器访问你的IP地址，就会看到Apache默认页面Ubuntu Logo Apache2 Ubuntu Default Page

sudo apt-get install php 安装php
sudo apt-get install libapache2-mod-php 安装php-mod-sgi
sudo apt-get install php-mysql 安装php-mysql

sudo apt-get install mysql-server 安装Mysql 不出意外会出现让你设置密码的界面，重复两遍即可。

sudo passwd 输入两次密码设置root账户的密码

配置数据库

输入 mysql -u root -p，然后输入密码进入数据库，这个密码是你刚才创建Mysql帐号的密码。

这样你就创建了一个数据库 wp
